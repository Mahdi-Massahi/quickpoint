We try the 'Hammer' approach: aggressive prompt engineering. We explicitly command the LLM to use tools for every word.

The results improve, but it's still not 100%. Some words are still missed. The LLM's nature is probabilistic, and no amount of "YOU MUST" will change that into 100% reliability.
