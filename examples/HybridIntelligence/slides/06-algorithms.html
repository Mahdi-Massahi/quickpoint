Example Problem: The Phonebook Search

Find "Zoe" in a phonebook of 1,000,000 names.

Linear Search üö≤

Check every name one by one starting from page 1.

1,000,000 steps

Binary Search üöÄ

Both solves the problem, but the choice of "algorithm" changes everything.

In computer science, an algorithm is a finite, step-by-step set of well-defined, unambiguous instructions used to solve a specific problem, perform calculations, or process data.

[[Slide Notes#Algorithms]]

Algorithms are the tools we use to solve problems.

If a Problem is a 'destination' we want to reach, an Algorithm is the 'vehicle' or the 'map' we use to get there.

Example: Phonebook search. Linear search is like a bicycle‚Äîit gets you there, but it's slow. Binary search is like a jet plane. The difference between 1 million steps and 20 steps is astronomical.
