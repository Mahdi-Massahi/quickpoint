[[Slide Notes#The Hybrid Intelligence]]

This is fantastic! This means we are now able to attempt to solve problems programmatically which we were unable to before LLMs.

The core philosophy: We shouldn't ask LLMs to solve the problem directly. We should ask them to translate the messy real-world Issue into a formal Problem that a classical algorithm can solve.

Offloading to deterministic algorithms gives us the most efficient, accurate, and deterministic results.
