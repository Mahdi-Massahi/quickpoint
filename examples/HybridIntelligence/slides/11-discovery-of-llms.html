[[Slide Notes#The LLM hammer]]

LLMs are amazing because they are flexible. We can use them as 'algorithms' for almost anything.

But there's a danger. When we replace a perfect, deterministic algorithm (P) with an LLM, we are moving the problem into BPP (probabilistic).

We are using a very flexible, but less certain, tool for tasks where we already have certainty.
